## OS support

- [x] Linux
- [x] MacOS
- [x] OpenBSD
- [x] FreeBSD
- [x] NetBSD
- [x] Haiku

## Installasi

Clone repositori dengan cara

```
git clone https://github.com/bellshade/bellfetch
cd bellfetch
sudo cp bellfetch /usr/bin
```
